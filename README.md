# Gacha Counter

Gacha pity counter for Genshin Impact / Honkai: Star Rail

## Prerequisites

- Bun v1.x

## Setup

1. Install dependencies
    ```bash
    bun install
    ```
2. Download static assets (game item data and images)
    ```bash
    bun dl-assets
    ```

## Development

Start the Nuxt development server
```bash
bun dev
```
